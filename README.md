# multi-image
A powerful image generation and editing plugin supporting multiple Al providers

一个支持文生图和图生图的Obsidian插件

支持添加多个文生图和图生图供应商，可以多轮对话，可以自定义提示词，可以自定义json参数。

支持六大生图服务商，包括:

智谱AI的glm-image

阿里云的qwen-image-2.0，qwen-image-2.0-pro和z-image-turbo

火山云的doubao-seedream-5.0，doubao-seedream-4.5和doubao-seedream-4.0

黑森林实验室BFL的flux2.0全系列五个

openai的gpt-image-1.5、gpt-image-1.0、gpt-image-1.0-mini的v1/generations格式和v1/edits格式

谷歌的v1beta格式的小香蕉2.0和大香蕉。

图片支持本地的图片上传，也支持单独粘贴url。

返回的文本和图片可以返回当前笔记，可以返回其他特定笔记，也可以新建笔记。

开始使用

Multi Image 目前尚未上架 Obsidian 社区插件商店，请按照以下步骤手动安装。

手动安装

前往 Releases 页面下载最新版本的zip

在你的 Vault 插件目录下创建文件夹：/.obsidian/plugins/obsidian-multi-image/

将下载的文件解压缩复制到该文件夹中

打开 Obsidian 设置 → 社区插件

在插件列表中启用 "Multi Image"

在插件设置中配置你的相关设置

注意事项

更多参数设置，以各个生图服务商官网为准。

url建议带jpg的长链。
